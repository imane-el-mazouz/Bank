# E-Bank Solution


## Contexte du Projet

La digitalisation des services bancaires est devenue incontournable pour offrir aux clients une expérience fluide et accessible. Cette application e-bank vise à fournir une plateforme sécurisée et intuitive permettant aux utilisateurs de gérer leurs comptes bancaires et d'effectuer des opérations financières en ligne.

## Objectifs du Projet

Développer une application bancaire complète qui inclut une interface utilisateur moderne et fonctionnelle ainsi qu'un backend robuste. Le projet se compose de deux parties :

1. **Backend :** Développé avec Spring Boot
2. **Frontend :** Développé avec Angular

## Technologies Utilisées

### Backend

- **Framework :** Spring Boot
- **Langage :** Java
- **Technologies :**
  - Spring MVC
  - Spring Security
  - Spring Data JPA
  - JUnit
  - MySQL/PostgreSQL
  - Docker (pour la containerisation)

### Frontend

- **Framework :** Angular 2+
- **Langage :** TypeScript
- **Styles :** SCSS, Bootstrap/TailwindCSS
- **Outils de Build :** Angular CLI

## Contexte Technique

### Backend (Spring Boot)

L'API backend est responsable de la gestion des comptes bancaires, des cartes, des transferts d'argent, et de l'authentification des utilisateurs. Elle fournit des endpoints RESTful pour interagir avec les données.

### Frontend (Angular)

Le frontend est une application monopage (SPA) qui communique avec l'API backend pour gérer les comptes, les cartes, les transferts, et l'authentification. L'interface utilisateur est construite avec Angular et stylisée avec SCSS et un framework CSS.
